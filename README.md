# Sama Apps

A simple static marketplace to discover and download Android apps built by independent developers.

**Live:** [https://sama-ndari.github.io/Sama-Apps](https://sama-ndari.github.io/Sama-Apps)

## Apps

| App | Category | Status |
|-----|----------|--------|
| **E-Sama** — Boutique & POS Management | Business | Available |
| **Velora** — Productivity & Wellness | Productivity | Available |
| **NotifSpy** — Notification Tracker | Utility | In Development |

## Features

- Browse and filter apps by category
- App detail pages with screenshots, descriptions, and download links
- Light / Dark mode
- English / French language toggle
- Fully responsive (mobile, tablet, desktop)
- APK downloads via Google Drive

## Tech

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step.
